# Scooter Rental Hypothesis Testing

This notebook analyzes user behavior and revenue for the GoFast scooter rental service. It includes exploratory analysis, data preparation, revenue calculation, and hypothesis testing to evaluate business questions about subscriptions and ride patterns.

## Project Contents
- Jupyter notebook with data loading, preprocessing, EDA, and statistical tests
- Visualizations of user distribution, subscription split, and ride duration
- Revenue model per user per month
- Hypothesis tests on ride duration, distance, and revenue

## Data Sources
The notebook loads datasets from:
- users_go.csv (users)
- rides_go.csv (rides)
- subscriptions_go.csv (pricing)
*Note: The raw datasets are proprietary property of the training provider and cannot be publically shared or distributed within this repository.*

## Key Steps
1. Load and inspect data
2. Clean and prepare tables
3. Explore user demographics and ride metrics
4. Merge datasets
5. Compute monthly revenue per user
6. Test business hypotheses with t-tests
7. Model distributions for ride duration and distance
